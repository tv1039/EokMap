# 억맵

👉 [앱 바로가기](여기에-배포링크-넣기)

목표 금액까지의 거리와 현재 재무 상태를 확인할 수 있는 개인용 자산 관리 웹 앱

## 소개

억맵은 수입, 자산, 고정지출 등의 정보를 입력하면  
현재 상황을 기준으로 목표까지의 방향을 확인할 수 있게 만든 앱입니다.

단순히 기록만 하는 가계부가 아니라  
내 상황을 직접 입력하고, 얼마나 모아야 하는지, 어떤 흐름으로 가야 하는지를  
한 번 더 현실적으로 점검해보는 용도로 만들었습니다.

한국, 일본, 미국 버전을 함께 넣어서  
각 나라 기준에 맞는 항목과 분위기로 확인할 수 있게 구성했습니다.

## 기능

- 국가 선택 지원
- 수입, 자산, 고정지출 입력
- 카테고리별 고정지출 추가 및 수정
- 현재 상황 기준 방향성 제안
- 목표 기준 스케줄표 표시
- 국가별 분위기를 반영한 구성

## 메모

내 재무 상황을 직접 확인하고  
과소비를 줄이면서 목표 금액까지 가기 위한 흐름을 점검하려고 만든 개인 프로젝트
